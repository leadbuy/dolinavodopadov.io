# Эко-этнопарк «Долина Водопадов» - Документация проекта

## Общая информация

Проект представляет собой веб-сайт для экологического парка "Долина Водопадов", расположенного в Карелии. Сайт представляет собой одностраничное веб-приложение, которое информирует посетителей о парке, его достопримечательностях, ценах, способах добраться и других важных деталях.

## Описание файлов и блоков

### HTML (index.html)

Сайт состоит из следующих основных секций:

1. **Шапка сайта (header)**
   - Логотип парка с названием "Долина Водопадов"
   - Бургер-меню для мобильных устройств
   - Навигация по сайту (О нас, Достопримечательности, Галерея, Отзывы, Как добраться, Цены)

2. **Главный экран (hero-section)**
   - Слайдер фоновых изображений с плавными переходами
   - Название парка с анимацией появления
   - Кнопки "Как до нас добраться" и "Стоимость билетов"

3. **Блок "О нас" (about-section)**
   - Описание парка и его концепции
   - Статистика (7 водопадов, 50+ гектаров территории, 5 тематических зон)

4. **Достопримечательности (attractions-section)**
   - 8 блоков с описанием различных достопримечательностей:
     - Экотропа
     - Каскад водопадов на реке Иййоки
     - Водяная мельница
     - Саамская деревня
     - Оленья ферма "Семь хуторов"
     - Кафе с традиционной кухней
     - Магазин сувениров и фермерских продуктов

5. **Галерея (gallery-section)**
   - Сетка изображений с возможностью просмотра в увеличенном виде
   - Попап с навигацией между изображениями

6. **Отзывы (reviews-section)**
   - Слайдер с фотографиями отзывов
   - Виджет Яндекс.Карт с отзывами

7. **Как добраться (directions-section)**
   - Инструкции по маршруту из Санкт-Петербурга и Сортавалы
   - Координаты парка
   - Встроенные карты
   - Кнопка построения маршрута в навигаторе

8. **Цены (prices-section)**
   - Три типа билетов:
     - Единый билет на все локации
     - Билет в оленью ферму "Семь хуторов"
     - Билет в усадьбу "Лумиваара"
   - Подробное описание включенных услуг
   - Информация о скидках и льготах

9. **Футер сайта (footer)**
   - Контактная информация
   - Часы работы
   - Социальные сети
   - Политика конфиденциальности

### CSS (styles.css)

Стилевой файл включает:

1. **CSS-переменные** для унификации цветов, шрифтов и других параметров
2. **Подключение кастомных шрифтов** (CoreMellow)
3. **Адаптивный дизайн** с поддержкой мобильных устройств
4. **Анимации и переходы** для улучшения пользовательского опыта
5. **Стили для всех блоков** сайта с подробной настройкой внешнего вида
6. **Медиа-запросы** для адаптации под разные размеры экранов

### JavaScript (script.js)

Скрипт реализует:

1. **Менеджер мобильного меню** - открытие/закрытие бургер-меню
2. **Слайдер изображений** - автоматическое и ручное переключение слайдов
3. **Видео-галерея** - управление воспроизведением видео
4. **Модальные окна** - форма обратной связи и окно с подробной информацией
5. **Валидация форм** - проверка корректности ввода данных
6. **Навигация** - плавная прокрутка к секциям
7. **Галерея** - просмотр изображений в попапе
8. **Карты** - интеграция с Яндекс.Картами
9. **Оптимизация производительности** - ленивая загрузка, троттлинг и др.

## Функциональные возможности

1. **Адаптивный дизайн** - сайт корректно отображается на устройствах с разным размером экрана
2. **Интерактивные элементы** - слайдеры, модальные окна, навигация
3. **Локализация** - весь контент на русском языке
4. **Доступность** - поддержка ARIA-атрибутов для улучшения доступности
5. **SEO-оптимизация** - использование семантических тегов и метаданных
6. **Производительность** - оптимизация загрузки ресурсов и ленивая загрузка изображений

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Яндекс.Карты API
- Адаптивная верстка
- Ленивая загрузка изображений
- Анимации и переходы

## Установка и запуск

Для запуска сайта достаточно открыть файл `index.html` в любом современном веб-браузере. Для корректной работы всех функций рекомендуется использовать сервер разработки или локальный веб-сервер.
