# Парк природы «Долина Водопадов»

Это веб-сайт для экологического парка «Долина Водопадов» в Карелии, представляющий парк, его достопримечательности, цены и информацию для посетителей.

## Описание функциональности

### Главная страница
- Адаптивный дизайн для всех устройств
- Анимированный фон с переключающимися изображениями
- Навигация по разделам сайта
- Кнопки для перехода к разделам "Как добраться" и "Цены"
- Динамическая загрузка контента из JSON-файлов

### Разделы сайта
1. **О нас** - информация о парке, его площади и достопримечательностях
2. **Достопримечательности** - подробное описание тематических зон с возможностью просмотра изображений
3. **Галерея** - интерактивная галерея с возможностью просмотра изображений в полном размере
4. **Отзывы** - фотографии посетителей с возможностью просмотра отзывов на Яндекс.Картах
5. **Как добраться** - инструкции по маршруту из разных направлений и карта с координатами
6. **Цены** - информация о стоимости билетов с детализацией по категориям

### Админ-панель
- Система аутентификации с логином/паролем (admin/admin по умолчанию)
- Управление контентом страницы (редактирование текстов)
- Управление достопримечательностями (создание, редактирование, удаление)
- Загрузка/удаление изображений для слайдера
- Загрузка/удаление изображений для достопримечательностей
- Загрузка/удаление изображений для галереи
- Изменение порядка отображения достопримечательностей
- Смена пароля администратора

### Технические особенности
- Полностью адаптивный дизайн
- Ленивая загрузка изображений для улучшения производительности
- Интерактивные галереи с автопереключением
- Модальные окна для дополнительной информации
- Интеграция с Яндекс.Картами
- Плавная навигация по странице
- Поддержка современных браузеров
- Доступность (a11y) элементов интерфейса
- Динамическая загрузка контента без перезагрузки страницы
- Управление контентом через JSON-файлы
- Безопасная загрузка изображений с проверкой формата и размера

### Технологии
- Python Flask (веб-фреймворк)
- HTML5
- CSS3 (с использованием CSS-переменных и Flexbox/Grid)
- JavaScript (ES6+)
- API Яндекс.Карт
- JSON (для хранения данных)
- Адаптивные изображения
- Семантическая разметка

## Запуск проекта

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Запустите сервер:
   ```
   python app.py
   ```

3. Откройте в браузере:
   ```
   http://localhost:5011
   ```

4. Для доступа к админ-панели перейдите:
   ```
   http://localhost:5011/login
   ```
   Логин: admin
   Пароль: admin
